🚀 Compilation Instructions

To compile the ISO natively on an Arch Linux terminal:

sudo chmod +x build.sh
sudo ./build.sh

To build using the container sandbox (from Windows or Debian hosts):

docker build -t aegis-builder .
docker run --privileged --rm -v ${PWD}:/build aegis-builder ./build.sh
